Quick Sort is a fast, divide-and-conquer sorting algorithm. It picks a pivot element and rearranges the array so that smaller elements are on the left and larger ones on the right.

In your implementation, the last element is used as the pivot.

üîÅ How It Works (Detailed)

Here‚Äôs the main Quick Sort function:

function quickSort(arr, low, high) {
  if (low < high) {
    let pi = partition(arr, low, high);
    quickSort(arr, low, pi - 1);
    quickSort(arr, pi + 1, high);
  }
}

Step-by-step explanation:

  1. Only sort when low < high.
  2. Call partition() to position the pivot correctly.
  3. Recursively sort left and right subarrays.

üìå Partition Logic (Explained with Code)

function partition(arr, low, high) {
  let pivot = arr[high];              // Step 1: Last element as pivot
  let i = low - 1;                    // Step 2: Pointer for smaller items

  for (let j = low; j < high; j++) {
    if (arr[j] < pivot) {            // Step 3: If element < pivot
      i++;
      [arr[i], arr[j]] = [arr[j], arr[i]];  // Step 4: Swap to left
    }
  }

  [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];  // Step 5: Place pivot
  return i + 1;
}

What‚Äôs happening

  • pivot = arr[high] ‚Üí Choose the last element as pivot.
  • i = low - 1 ‚Üí Keeps track of where smaller items should go.
  • Compare all elements from low to high - 1.
  • Swap smaller items to the left side.
  • Finally, move the pivot to the middle.

‚úÖ Result: Elements left of pivot are smaller, right are larger ‚Äî and the pivot is correctly placed!
